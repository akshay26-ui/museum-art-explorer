## ✦ What Is This? ✦

</div>

> The Metropolitan Museum of Art holds over 470,000 artworks spanning 5,000 years of human history — and now you can explore all of it from your bedroom. **Museum Art Explorer** lets you search by artist, filter by era, save your favorites, and stumble upon something beautiful you never knew existed. World-class art, zero tickets required.

│       ├── Artwork cards with hover effect
│       └── Loading spinner + error message
│
├── ⚙️   script.js      ← All logic
│       ├── search()      fetch artwork IDs from API
│       ├── loadPage()    fetch and display 12 artworks
│       ├── sortList()    sort results using .sort()
│       ├── makeCard()    build each artwork card
│       ├── toggleFav()   save / remove favorites
│       └── buildPages()  previous / next page buttons
│
